Use this guide when a model or research table needs normalized statements, tag-level facts, and the filing context needed to reconcile them. It is not a replacement for issuer-specific accounting judgment, taxonomy review, or a restatement policy.

## The buyer job

Refresh one issuer's annual financial inputs while retaining enough evidence for a model reviewer to reconcile a statement line to its fact series. Choose raw SEC XBRL if your team needs to own taxonomy parsing and normalization. Choose this path when a reusable normalized layer plus provenance is the practical requirement.

## Smallest proof of concept

Retrieve one annual statement bundle and a small annual fact series, then compare only aligned periods and units.

```bash
curl --fail-with-body -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/statements/all?ticker=MSFT&period=annual&limit=1"

curl --fail-with-body -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/facts?ticker=MSFT&tag=Revenues&form=10-K&limit=4"
```

For the selected revenue row, compare `form`, fiscal labels, `periodEnd`, `filedAt`, unit, and the fact response's requested and resolved tag before treating values as reconcilable.

## What a passing result contains

For every model input, preserve requested tag, `resolvedTag`, value, unit, form, fiscal period, period end, filing date, and provenance with its source or filing URL. Preserve `accessionNumber` when it is present; it can be null for a valid response. Also retain enclosing `requestId`, `traceparent`, and freshness where supplied. Values are raw units; display conversion belongs in an explicit model layer.

## Source and production limits

- Normalization does not remove issuer extensions, taxonomy variation, amendments, restatements, or period-basis choices.
- A missing tag is not proof that an issuer did not disclose the metric; inspect the resolved concept and the issuer's reporting context.
- Annual and quarterly values are not interchangeable.
- Reconciliation failures should first be checked for units, form, period, filing selection, and restatement policy.
- Cache historical observations, bound refresh concurrency, and quote the actual issuer cadence through [Pricing and limits](/evaluate/pricing-and-limits).

## Next decision

Adopt this layer for a model only after a reviewer can reconcile one chosen statement line to the relevant fact series and document the exception policy for the issuer types you cover.
