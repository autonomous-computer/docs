---
title: "13F API for institutional holdings"
description: "Retrieve a manager's reported 13F holdings, select a filed period, and preserve the evidence needed to interpret a delayed disclosure"
---

Call the 13F routes when you need a normalized report for an institutional manager rather than a live portfolio estimate. Start with a manager CIK, then retain the report date, filing date, accession number, and filing URL in the result you show a user or pass to an agent.

## Get the latest reported holdings

```bash
curl -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/owners/13f?cik=0001067983&limit=20"
```

The selected report includes the manager, filing and report dates, the accession number, holdings, and provenance. `limit` is capped at 50. To return the compact, citation-preserving shape for an agent, add `view=agent`.

## Select a reported period

First list the manager's available 13F filings:

```bash
curl -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/owners/13f/filings?cik=0001067983&limit=8"
```

Then select a report with a returned date:

```bash
curl -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/owners/13f?cik=0001067983&reportDate=2024-12-31&limit=20"
```

`reportDate` is the quarter-end report date; `filingDate` is when the SEC received the filing. Do not substitute one for the other in a screen, report, or alert.

## Compare latest available reports

Use the comparison route to obtain a row-level change list for the latest two parsable reports:

```bash
curl -X POST \
  -H "content-type: application/json" \
  -H "x-api-key: $SECAPI_API_KEY" \
  -d '{"cik":"0001067983","limit":50}' \
  "https://api.secapi.ai/v1/owners/13f/compare"
```

This endpoint does not compare user-selected periods. Its `added`, `removed`, and `changed` rows identify differences in the two selected disclosures; they do not establish when a manager traded, why it changed the position, or what it owns now.

## Build with disclosure limits

- Form 13F is a periodic disclosure, not a real-time position feed.
- It has its own reporting scope and does not show a manager's complete exposure, including every short, derivative, cash, liability, or non-reportable asset.
- Reported value is not a current market value.
- An amendment, manager identity change, or security-class detail can change the reading of a row. Use the accession number and filing URL to inspect the filing.

For issuer-centric reported holders, use `GET /v1/owners/institutional/ticker` and label the returned cohort's dates.
