## Search an issuer's filing history

```bash
curl -sS -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/filings?ticker=NVDA&form=10-K&limit=5&sort=filing_date_desc"
```

For one issuer's newest annual report, use `GET /v1/filings/latest?ticker=NVDA&form=10-K`. Without `form`, latest filing means the newest filing of any type. For a known filing, use the accession-number route. For a question about content, use a content-search route rather than treating the manifest as the answer.

## Continue with the filing you found

```bash
curl -sS -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/filings/0001045810-24-000029"

curl -sS -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/filings/0001045810-24-000029/sections/item_1a?ticker=NVDA&form=10-K"
```

Keep `accessionNumber`, `filingDate`, `filingUrl`, and `requestId` with any excerpt or report. Preserve route-specific provenance and freshness metadata when supplied.

## Choose the right route

| Need                            | Endpoint                                                                                       |
| ------------------------------- | ---------------------------------------------------------------------------------------------- |
| Filter filing records           | [`GET /v1/filings`](/api-reference/filings/get-v1-filings)                                     |
| Find one issuer's newest filing | [`GET /v1/filings/latest`](/api-reference/filings/get-v1-filings-latest)                       |
| Read a known filing             | [`GET /v1/filings/{accession_number}`](/api-reference/filings/get-v1-filings-accession-number) |
| Search exact text               | [`GET /v1/search/fulltext`](/api-reference/search/get-v1-search-fulltext)                      |
| Search related concepts         | [`GET /v1/search/semantic`](/api-reference/search/get-v1-search-semantic)                      |
| Search cited passages           | [`GET /v1/sections/search`](/api-reference/sections/get-v1-sections-search)                    |
