# Platform introspection

SEC API exposes lightweight operational surfaces for builders:

- `GET /v1/usage`
- `GET /v1/limits`
- `GET /v1/delivery/events`

## Usage

`/v1/usage` summarizes locally logged usage events for the current org by meter class.

## Limits

`/v1/limits` returns the current quota policy table by meter class.

## Events

`/v1/delivery/events` returns recent tenant-scoped records from the durable Delivery ledger: canonical events, webhook delivery attempts, and stream delivery records. New integrations should use the delivery namespace for delivery history so it does not collide with SEC event APIs.

Supported filters:

- `kind`
- `type`
- `limit`
