# Platform introspection

Omni Datastream exposes lightweight operational surfaces for builders:

- `GET /v1/usage`
- `GET /v1/limits`
- `GET /v1/events`

## Usage

`/v1/usage` summarizes locally logged usage events for the current org by meter class.

## Limits

`/v1/limits` returns the current quota policy table by meter class.

## Events

`/v1/events` returns recent platform events from the local event log.

Supported filters:

- `kind`
- `type`
- `limit`
