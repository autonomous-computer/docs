# Changelog

Public release notes for SEC API. Entries focus on behavior, new endpoints, compatibility changes, and documentation updates.

<Update label="June 24, 2026" description="SDK 1.0.1 release train" tags={["SDKs"]}>

- Published `1.0.1` packages for JavaScript, Python, Go, Rust, and the CLI from mirror repositories checked against the monorepo SDK source.
- Verified clean-room installs from npm, PyPI, crates.io, and the Go module tag before marking the release ready.
- Updated the Homebrew tap to install the published `@secapi/cli@1.0.1` tarball.

</Update>

<Update label="May 20, 2026" description="Docs renamed to SEC API" tags={["Docs"]}>

- Renamed the documentation site, navigation, OpenAPI title, examples, and LLM entrypoint to SEC API.
- Updated public navigation around getting started, API reference, use cases, status, and trust.
- Updated the Mintlify API playground configuration and public OpenAPI file path.
- Added a glossary and LLM/agent guide.

</Update>

<Update label="April 27, 2026" description="Public pricing catalog" tags={["Billing", "Pricing"]}>

- Published `GET /v1/billing/rates` as the source of truth for public plan metadata and meter-family rates.
- Added Pay As You Go, Personal, Team, and Commercial plan metadata.
- Added public meter prices for light reads, standard reads, heavy extracts, artifact jobs, delivery events, intelligence queries, market data reads, and semantic search.

</Update>

<Update label="April 26, 2026" description="Dilution score and enhanced dilution signal" tags={["Signals", "Dilution"]}>

- Added `GET /v1/signals/dilution` for issuer-level dilution risk scoring.
- Added `GET /v1/signals/dilution/enhanced` for richer factor context and explanation metadata.
- Added dilution-specific documentation and examples for events, warrants, convertibles, lockups, cash position, corporate actions, Nasdaq compliance, ratings, reverse splits, share float history, and coverage.

</Update>

<Update label="April 24, 2026" description="Market data and index utilities" tags={["Market Data"]}>

- Added market calendar, market bars, market snapshots, market search, corporate actions, estimates, earnings calendar, reference, index inventory, and index constituent documentation.
- Clarified rights and coverage boundaries for market data routes.
- Clarified rights and coverage boundaries for public index constituent exports.

</Update>

<Update label="April 22, 2026" description="AskEdgar migration and dilution comparison" tags={["Migration", "Dilution"]}>

- Added AskEdgar migration guidance for dilution workflows.
- Added pricing and schema comparison notes using public AskEdgar pricing and API references.
- Clarified verification metadata differences and cutover considerations.

</Update>

<Update label="April 18, 2026" description="Webhooks, streams, and monitors" tags={["Webhooks", "Streams"]}>

- Added monitor lifecycle documentation for saved searches and alert delivery.
- Added webhook endpoint, delivery history, replay, and signing secret rotation docs.
- Added stream subscription and cursor polling docs for replay-safe event delivery.

</Update>

<Update label="April 12, 2026" description="Macro, factors, and intelligence workflows" tags={["Macro", "Factors", "Intelligence"]}>

- Added macro calendar, indicators, regimes, releases, forecasts, search, and high-signal pack documentation.
- Added factor catalog, returns, exposures, correlations, dashboards, and strategy workflow documentation.
- Added intelligence endpoints for company, security, portfolio, country report, earnings preview, footnotes query, and watchlist workflows.

</Update>
