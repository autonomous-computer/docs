# Install in your client

SEC API's hosted MCP server speaks streamable HTTP and works with MCP-aware clients. One URL and one auth header expose the SEC API tool set.

```text
URL:    https://api.secapi.ai/mcp
Header: x-api-key: $SECAPI_API_KEY
```

Pick your client below. Each page is a copy-paste install with a "verify" prompt that confirms tools are wired up.

## IDE-native MCP clients

- [Claude Desktop](/mcp-install#claude-desktop) — config in `claude_desktop_config.json`
- [Claude Code](/mcp-install#claude-code) — single `claude mcp add` command
- [Cursor](/mcp-install#cursor) — config in `.cursor/mcp.json`
- [Windsurf](/clients/windsurf) — config in `~/.codeium/windsurf/mcp_config.json`
- [Continue](/clients/continue) — config in `~/.continue/config.yaml`

## Agent frameworks

- [LangChain](/clients/langchain) — Python and JavaScript via `langchain-mcp-adapters`
- [LlamaIndex](/clients/llamaindex) — Python via `llama-index-tools-mcp`
- [Vercel AI SDK](/clients/vercel-ai-sdk) — TypeScript via `@ai-sdk/mcp`

## Hosted assistants

- [Custom GPT](/clients/custom-gpt) — MCP connector where available, plus an OpenAPI Action fallback

## Don't see your client?

For clients that support MCP over streamable HTTP, point the server at `https://api.secapi.ai/mcp` and pass `x-api-key: $SECAPI_API_KEY`. The full reference config lives on [MCP Server Installation](/mcp-install).

Create an API key at [secapi.ai](https://secapi.ai), then pass it as `x-api-key: $SECAPI_API_KEY`.
