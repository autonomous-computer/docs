## Example

```go
package main

import (
	"fmt"
	"io"
	"net/http"
	"net/url"
	"os"
)

func main() {
	params := url.Values{"ticker": {"AAPL"}}
	req, err := http.NewRequest("GET", "https://api.secapi.ai/v1/entities/resolve?"+params.Encode(), nil)
	if err != nil {
		panic(err)
	}
	req.Header.Set("x-api-key", os.Getenv("SECAPI_API_KEY"))

	res, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer res.Body.Close()

	body, err := io.ReadAll(res.Body)
	if err != nil {
		panic(err)
	}
	fmt.Println(string(body))
}
```

Use these direct REST examples when you want explicit control over retries, logging, and response metadata in Go.

## Dilution intelligence REST calls

```go
// Top 10 highest-dilution-risk issuers
req, err := http.NewRequest("GET", "https://api.secapi.ai/v1/dilution/ratings?overall_risk=high&limit=10", nil)
req.Header.Set("x-api-key", apiKey)
res, err := http.DefaultClient.Do(req)

// Single-ticker dilution rollup
req, err = http.NewRequest("GET", "https://api.secapi.ai/v1/dilution/score?ticker=BBBB", nil)
req.Header.Set("x-api-key", apiKey)
res, err = http.DefaultClient.Do(req)

// ATM offerings only (boolean filter on the events list)
req, err = http.NewRequest("GET", "https://api.secapi.ai/v1/dilution/events?ticker=BBBB&is_atm=true", nil)
req.Header.Set("x-api-key", apiKey)
res, err = http.DefaultClient.Do(req)

// Coverage + freshness rollup (no agent shape — already compact)
req, err = http.NewRequest("GET", "https://api.secapi.ai/v1/dilution/coverage?ticker=BBBB", nil)
req.Header.Set("x-api-key", apiKey)
res, err = http.DefaultClient.Do(req)
```

The full dilution API is available under `/v1/dilution/*`. Pass `view=agent` in list-call query params, except `/v1/dilution/coverage`, for the compact agent-mode shape.
