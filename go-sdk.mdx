# Go SDK

<Info>
The Go SDK is deliberately thin. It mirrors the canonical REST contract and stays useful when you want explicit request control without building a generated client jungle.
</Info>

## Example

```go
package main

import (
	"fmt"
	"io"
	"net/http"
	"net/url"
	"os"
)

func main() {
	params := url.Values{"ticker": {"AAPL"}}
	req, err := http.NewRequest("GET", "https://api.secapi.ai/v1/entities/resolve?"+params.Encode(), nil)
	if err != nil {
		panic(err)
	}
	req.Header.Set("x-api-key", os.Getenv("SEC_API_KEY"))

	res, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer res.Body.Close()

	body, err := io.ReadAll(res.Body)
	if err != nil {
		panic(err)
	}
	fmt.Println(string(body))
}
```

When the Go module is published, prefer the public module path:

```bash
go get go.secapi.dev/secapi
```

## Dilution intelligence SDK shape

```go
// Top 10 highest-dilution-risk issuers
ratings, err := client.DilutionRatings(map[string]string{
  "overall_risk": "high",
  "limit":        "10",
})

// Single-ticker dilution rollup
score, err := client.DilutionScore(map[string]string{"ticker": "BBBB"})

// ATM offerings only (boolean filter on the events list)
atms, err := client.DilutionEvents(map[string]string{
  "ticker": "BBBB",
  "is_atm": "true",
})

// Coverage + freshness rollup (no agent shape — already compact)
coverage, err := client.DilutionCoverage(map[string]string{"ticker": "BBBB"})
```

The full surface — `DilutionEvents`, `DilutionEventDetail`, `DilutionWarrants`, `DilutionConvertibles`, `DilutionRofr`, `DilutionLockups`, `DilutionCashPosition`, `DilutionCorporateActions`, `DilutionNasdaqCompliance`, `DilutionRatings`, `DilutionReverseSplits`, `DilutionScore`, `DilutionShareFloatHistory`, `DilutionCoverage` — mirrors the REST routes 1:1. Pass `"view": "agent"` in any list call (except `DilutionCoverage`) for the compact agent-mode shape.
