# SEC API Catalog

The canonical machine-readable SEC API catalog is available as JSON at [docs.secapi.ai/.well-known/api-catalog.json](https://docs.secapi.ai/.well-known/api-catalog.json).

Use that JSON catalog for automation, OpenAPI discovery, SDK routing, MCP setup, API-family inventory, evaluation-page discovery, and agent workflow planning. The catalog points to the canonical OpenAPI spec at [/openapi/sec-api-public.v1.json](/openapi/sec-api-public.v1.json), the compatibility alias at [/openapi/sec-api.v1.json](/openapi/sec-api.v1.json), hosted MCP docs, marketing catalogs, and the docs evaluation pages.

This page exists as a crawler-safe fallback for clients that first request `/.well-known/api-catalog`.
