# SEC API Catalog

The canonical machine-readable SEC API catalog is available as JSON at [/.well-known/api-catalog.json](/.well-known/api-catalog.json).

Use that JSON catalog for automation, OpenAPI discovery, SDK routing, MCP setup, API-family inventory, and agent workflow planning. This page exists as a crawler-safe fallback for clients that first request `/.well-known/api-catalog`.
