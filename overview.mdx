## How to use these docs

<Steps>
  <Step title="Get one answer you can inspect">
    Resolve an issuer and retrieve a filing before you reach for a broad search or an agent loop.
  </Step>
  <Step title="Choose the smallest useful interface">
    REST is the shortest path to understanding a response. Use an SDK, CLI, or MCP once that contract is familiar.
  </Step>
  <Step title="Keep the evidence with the conclusion">
    Accession numbers, source URLs, filing dates, and response metadata make research reproducible when a result is revisited.
  </Step>
</Steps>
