## Start in three steps

<Steps>
  <Step title="Resolve one real company">
    Start with a company you already know. Entity resolution is the fastest way to confirm auth, identifier support, freshness metadata, and the general contract shape.
  </Step>
  <Step title="Run one filing or statements workflow">
    Pull the latest filing, extract a section, fetch the all-statements bundle, or hydrate a trace. Do one real investor task instead of a synthetic ping.
  </Step>
  <Step title="Give the workflow to your agent">
    Copy a prompt from the docs and hand it to your coding agent so the integration begins with a useful end-to-end task that preserves provenance and traceability.
  </Step>
</Steps>

## Give this prompt to your agent

Wire Omni Datastream into this project.
