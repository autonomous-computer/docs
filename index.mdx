# Omni Datastream Docs

Omni Datastream is the production SEC data plane behind `api.secapi.ai`.

## Latest platform update

- docs are now live on `docs.secapi.ai`
- Omni Datastream `2.0` core execution is complete
- broad historical coverage is now in place for the highest-value filing families:
  - `10-K`
  - `10-Q`

  - `13F`
- the manifest-first section path now clears the final live acceptance benchmark:
  - `section_item_1a_aapl`: Omni `155.36ms` p50 vs `sec-api` `295.79ms` p50
- native MCP install flow and first-class SDKs are live

## Start here

- Base API: `https://api.secapi.ai`
- Auth today: `x-api-key`
- Hosted MCP: `https://api.secapi.ai/mcp`
- API health: `https://api.secapi.ai/healthz`

## Core workflows

- resolve issuers by ticker or CIK
- search filings and sections
- retrieve facts and statements
- compare 13F and compensation data
- generate artifacts, events, webhooks, and stream feeds
