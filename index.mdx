# Omni Datastream Docs

Omni Datastream is the production SEC data plane behind `api.secapi.ai`.

## Start here

- Base API: `https://api.secapi.ai`
- Auth today: `x-api-key`
- Hosted MCP: `https://api.secapi.ai/mcp`
- API health: `https://api.secapi.ai/healthz`

## Core workflows

- resolve issuers by ticker or CIK
- search filings and sections
- retrieve facts and statements
- compare 13F and compensation data
- generate artifacts, events, webhooks, and stream feeds
