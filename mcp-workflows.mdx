# Hosted MCP Workflows

## Why use hosted MCP

Hosted MCP lets coding agents call the same Datastream workflows directly instead of forcing you to wrap every path yourself.

## Install metadata

```bash
curl https://api.secapi.ai/mcp
```

That payload includes:

- the canonical SSE transport URL
- auth metadata links
- install hints
- current tool and resource summaries

## Common clients

### Claude Code

```bash
claude mcp add --transport sse omni-datastream https://api.secapi.ai/mcp
```

### Cursor

```json
{
  "omni-datastream": {
    "url": "https://api.secapi.ai/mcp",
    "headers": {
      "x-api-key": "${OMNI_DATASTREAM_API_KEY}"
    }
  }
}
```
