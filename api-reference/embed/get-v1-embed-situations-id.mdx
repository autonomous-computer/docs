---
title: "GET /v1/embed/situations/{id}"
description: "Return public detail and events for a recent SEC-derived situation"
---

Return public detail and its event timeline for one recent SEC-derived situation; no API key is required.

## Example request

<RequestExample>
```bash
curl "https://api.secapi.ai/v1/embed/situations/sit_example"
```
</RequestExample>

## Public provenance

When present, `publicProvenance` identifies source filing accessions and may include public SEC document links or citation spans. It never includes internal telemetry, storage keys, extraction traces, model metadata, or confidence signals.

## Response and failures

Returns a JSON situation object with public events. Event records may include an `accessionNumber` and public SEC `documentUrl`.

A missing identifier returns `400`; an unknown, non-public, or no-longer-recent situation returns `404`; unexpected failures return `500`.
