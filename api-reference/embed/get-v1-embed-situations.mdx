---
title: "GET /v1/embed/situations"
description: "Return a public, recent-only list of SEC-derived situations"
---

Return a public, recent-only list of SEC-derived situations without an API key.

## Example request

<RequestExample>
```bash
curl "https://api.secapi.ai/v1/embed/situations?limit=20"
```
</RequestExample>

## Response and failures

Returns a JSON list envelope containing public situation summaries, pagination state, and a generation timestamp. Use `limit`, `cursor`, `types`, `statuses`, `tickers`, `sectors`, and `market_cap` to narrow the recent public window. An invalid cursor returns `400`; a disabled public embed surface returns `404`; unexpected failures return `500`.
