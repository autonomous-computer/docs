---
title: "GET /v1/embed/situations/feed.rss"
description: "Return a public RSS feed of recent situation events"
---

Return a public RSS feed of recent SEC-derived situation events without an API key.

## Example request

<RequestExample>
```bash
curl "https://api.secapi.ai/v1/embed/situations/feed.rss"
```
</RequestExample>

## Response and failures

Returns RSS XML with the `application/rss+xml` content type. Use `types`, `categories`, and `tickers` to filter the recent public feed. A disabled public embed surface returns `404`; unexpected failures return `500`.
