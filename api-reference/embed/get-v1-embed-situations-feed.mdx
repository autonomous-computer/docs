---
title: "GET /v1/embed/situations/feed"
description: "Return a public, recent-only feed of situation events"
---

Return a public, recent-only feed of SEC-derived situation events without an API key.

## Example request

<RequestExample>
```bash
curl "https://api.secapi.ai/v1/embed/situations/feed?limit=20"
```
</RequestExample>

## Response and failures

Returns a JSON list envelope containing public feed items, pagination state, and a generation timestamp. Use `limit`, `cursor`, `types`, `categories`, and `tickers` to filter the recent public window. An invalid cursor returns `400`; a disabled public embed surface returns `404`; unexpected failures return `500`.
