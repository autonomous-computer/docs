---
title: "GET /v1/embed/situations/stats"
description: "Return public aggregate statistics for recent SEC-derived situations"
---

Return public aggregate statistics for recent SEC-derived situations without an API key.

## Example request

<RequestExample>
```bash
curl "https://api.secapi.ai/v1/embed/situations/stats"
```
</RequestExample>

## Response and failures

Returns a JSON statistics object for the recent public window, including aggregates by situation type, status, sector, and market-cap grouping. A disabled public embed surface returns `404`; unexpected failures return `500`.
