Public Markdown Copy-for-LLM brief for one recent Special Situation

<Info>
Audience: application and coding agent. This is an anonymous public route. The response is Markdown, not a JSON envelope.
</Info>

## Canonical metadata

- `Content-Type: text/markdown; charset=utf-8`
- `Cache-Control: public, max-age=120`

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "secapi-version: 2026-03-19" \
  "https://api.secapi.ai/v1/embed/situations/sit_0123456789abcdef0123/export"
```
</RequestExample>

## Give this prompt to your agent

<Prompt>
Use SEC API GET /v1/embed/situations/{situation_id}/export to retrieve the anonymous, recent-window Copy-for-LLM brief. Preserve the Markdown body and its cited filing links; do not parse it as a JSON envelope or fill missing facts with guesses.
</Prompt>

## Failure posture

- treat `404` as an unknown, unavailable, or no-longer-recent public situation; do not invent a brief
- preserve successful response bodies as Markdown, not JSON
- retry `429` and `5xx` responses with bounded backoff
