---
title: "GET /v1/embed/situations/{id}/export"
description: "Return a public Markdown export for a recent SEC-derived situation"
---

Return a public Markdown export of one recent SEC-derived situation and its filing timeline; no API key is required.

## Example request

<RequestExample>
```bash
curl "https://api.secapi.ai/v1/embed/situations/sit_example/export"
```
</RequestExample>

## Public provenance

The export cites source filing accessions and includes public SEC links when available. It does not expose internal telemetry, storage keys, extraction traces, model metadata, confidence signals, or private citation data.

## Response and failures

Returns `text/markdown` containing the situation summary, timeline, and available source citations.

A missing identifier returns `400`; an unknown, non-public, or no-longer-recent situation returns `404`; unexpected failures return `500`.
