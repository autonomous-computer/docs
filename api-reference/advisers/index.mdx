# Advisers API

<Info>
Search SEC Form ADV adviser records and adviser metadata.
</Info>

## Use This API To

- Find registered investment advisers by name, CIK, or state.
- Add adviser lookup to diligence, compliance, or fund workflows.
- Keep adviser results tied to SEC Form ADV source records.
