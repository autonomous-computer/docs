# Signals API

<Info>
Retrieve derived public-market signals such as volatility and dilution risk.
</Info>

## Use This API To

- Read the SEC API Volatility Score and factor breakdown.
- Retrieve issuer-level dilution signals and enhanced context.
- Use signal freshness and degraded reasons before making decisions.

## Before You Call

## Endpoints

<CardGroup cols={2}>
<Card title="GET /v1/signals/volatility" icon="square-terminal" href="/api-reference/signals/get-v1-signals-volatility">Return the SEC API volatility score with transparent factor breakdown and freshness metadata</Card>
<Card title="GET /v1/signals/dilution" icon="square-terminal" href="/api-reference/signals/get-v1-signals-dilution">Return the SEC API Dilution Score: factor-weighted 0-100 derived metric with sub-ratings, percentile, and rolling history</Card>
