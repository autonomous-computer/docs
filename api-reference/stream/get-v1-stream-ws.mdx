# GET /v1/stream/ws

Upgrade to a WebSocket connection for real-time filing event streaming

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
npx wscat -c "wss://api.secapi.ai/v1/stream/ws?ticket=$SEC_API_STREAM_TICKET" \
  -H "secapi-version: 2026-03-19"

# Mint the short-lived ticket first:
curl -X POST \
  -H "x-api-key: $SEC_API_KEY" \
  -H "secapi-version: 2026-03-19" \
  -H "content-type: application/json" \
  -d '{}' \
  "https://api.secapi.ai/v1/stream/tickets"
```
</RequestExample>

<Info>
Mint the short-lived `$SEC_API_STREAM_TICKET` first with `POST /v1/stream/tickets` so you do not expose a long-lived API key in a browser-visible WebSocket URL.
</Info>
