# Stream Tickets API

<Info>
Mint short-lived tickets and connect to WebSocket event streams.
</Info>

## Use This API To

- Mint signed WebSocket stream tickets.
- Connect clients to real-time filing event streams.
- Keep stream auth short-lived and scoped to the current principal.
