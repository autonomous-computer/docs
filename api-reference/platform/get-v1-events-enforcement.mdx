Audience: application and coding agent.

## Agent mode

Pass `?view=agent` to receive the compact agent shape: release metadata (sourceType, publishedAt, releaseNumber, title, excerpt) + citation URLs (`documentUrl`, `htmlUrl`). The `summaryMd` prose digest (~3–6kb per action), capability, statusNote, trace, and metadata envelopes are dropped. Typical reduction: **~85%**. See [api conventions → response formats](/api-conventions) for the full `?view=` contract.

## Canonical metadata

- `requestId`
- `traceparent`
