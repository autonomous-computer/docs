# POST /v1/billing/credits/topup

Create a Stripe PaymentIntent to purchase prepaid credits, returning the client secret and the discount/fee charge breakdown

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
curl -X POST \
  -H "Authorization: Bearer $SECAPI_BEARER_TOKEN" \
  -H "secapi-version: 2026-03-19" \
  -H "content-type: application/json" \
  -d '{"amountUsd":50,"paymentMethodType":"us_bank_account"}' \
  "https://api.secapi.ai/v1/billing/credits/topup"
```
</RequestExample>

<Info>
Credit top-ups require a WorkOS bearer session with `billing:manage` and are limited to pay-as-you-go organizations with prepaid credits enabled. API keys are rejected, and the body must include a positive `amountUsd`.
</Info>
