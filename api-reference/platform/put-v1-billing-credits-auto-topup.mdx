# PUT /v1/billing/credits/auto-topup

Enable or disable automatic prepaid-credit top-ups and set the low-balance threshold and refill amount

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
curl -X PUT \
  -H "Authorization: Bearer $SECAPI_BEARER_TOKEN" \
  -H "secapi-version: 2026-03-19" \
  -H "content-type: application/json" \
  -d '{}' \
  "https://api.secapi.ai/v1/billing/credits/auto-topup"
```
</RequestExample>

<Info>
Billing funding changes require a WorkOS bearer session with `billing:manage`. API keys are rejected so payment setup, refunds, and auto-top-up controls stay limited to interactive billing administrators.
</Info>
