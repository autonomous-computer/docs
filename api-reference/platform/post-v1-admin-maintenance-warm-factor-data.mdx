# POST /v1/admin/maintenance/warm-factor-data

Warm operator-scoped factor returns, exposures, and correlations for covered symbols and factor categories

<Info>
Audience: operator.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
curl -X POST \
  -H "x-api-key: $OMNI_DATASTREAM_API_KEY" \
  -H "omni-version: 2026-03-19" \
  -H "content-type: application/json" \
  -d '{"refreshLive": false}' \
  "https://api.secapi.ai/v1/admin/maintenance/warm-factor-data"
```
</RequestExample>

`refreshLive=false` runs the warm path against already-materialized market data instead of forcing live provider refresh. Use that for deterministic full-universe backfills and repair runs.
