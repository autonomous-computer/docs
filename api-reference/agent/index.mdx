# Agent Bootstrap API

<Info>
Create short-lived sponsor tokens and exchange them for org-scoped agent credentials.
</Info>

## Use This API To

- Bootstrap MCP-aware agents without exposing long-lived human credentials.
- Issue single-use sponsor tokens from a human-authenticated organization session.
- Return API key, billing, limit, and MCP install metadata in one setup flow.
