<Info>
Audience: application and coding agent.
</Info>

- `requestId`
- `id`
- `tokenPrefix`
- `scopes`

## Example request

<RequestExample>
```bash
curl -X POST \
  -H "Authorization: Bearer $SECAPI_BEARER_TOKEN" \
  -H "secapi-version: 2026-03-19" \
  -H "content-type: application/json" \
  -d '{"label":"research-agent","scopes":["read:sec"],"ttlSeconds":900}' \
  "https://api.secapi.ai/v1/agent/bootstrap_tokens"
```
</RequestExample>

## Authentication

This is a human organization action. Send a bearer token for a signed-in organization member. API keys are rejected. The sponsor-token secret is returned once; do not log it.
