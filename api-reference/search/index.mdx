# Search API

<Info>
Run full-text and semantic search across SEC filing content and section text.
</Info>

## Use This API To

- Find filing content by exact keywords or semantic similarity.
- Retrieve citation-ready filing-section results.
- Use search results as inputs to section, filing, or intelligence workflows.
