# News API

<Info>
Search rights-safe news stories and issuer communications connected to public-market workflows.
</Info>

## Use This API To

- Search news by symbol, entity, or topic.
- Retrieve rights-safe story records with entity tagging.
- Connect news context to intelligence, portfolio, or monitor workflows.
