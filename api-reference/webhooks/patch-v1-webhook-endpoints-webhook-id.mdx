# PATCH /v1/webhook_endpoints/{webhook_id}

Update a webhook endpoint URL, description, subscribed event types, or active status

<Info>
Audience: application and admin.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
curl -X PATCH \
  -H "Authorization: Bearer $SECAPI_BEARER_TOKEN" \
  -H "secapi-version: 2026-03-19" \
  -H "content-type: application/json" \
  -d '{"destinationUrl":"https://example.com/hooks/secapi","description":"Production monitor matches","subscribedEventTypes":["monitor.match","webhook.test"],"status":"active"}' \
  "https://api.secapi.ai/v1/webhook_endpoints/wh_example_123"
```
</RequestExample>
