# POST /v1/webhook_endpoints

Create a signed webhook endpoint for artifact and stream events

<Info>
Audience: application.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
curl -X POST \
  -H "x-api-key: $SEC_API_KEY" \
  -H "secapi-version: 2026-03-19" \
  -H "content-type: application/json" \
  -d '{}' \
  "https://api.secapi.ai/v1/webhook_endpoints"
```
</RequestExample>
