# Artifacts

Artifact creation, retrieval, manifests, and download/reconcile workflows.

## Endpoints

<Card title="GET /v1/artifacts" icon="square-terminal" href="/api-reference/artifacts/get-v1-artifacts">List canonical artifacts for the current organization</Card>

<Card title="GET /v1/artifacts/{artifact_id}" icon="square-terminal" href="/api-reference/artifacts/get-v1-artifacts-artifact-id">Fetch persisted artifact metadata</Card>
<Card title="GET /v1/artifacts/{artifact_id}/download" icon="square-terminal" href="/api-reference/artifacts/get-v1-artifacts-artifact-id-download">Download or redirect to the persisted artifact payload</Card>

<Card title="GET /v1/artifacts/{artifact_id}/manifest" icon="square-terminal" href="/api-reference/artifacts/get-v1-artifacts-artifact-id-manifest">Fetch the structured manifest for a persisted artifact</Card>
