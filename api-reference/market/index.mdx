# Market Data API

<Info>
Retrieve market calendars, bars, snapshots, estimates, reference data, corporate actions, indices, and ticker universe records.
</Info>

## Use This API To

- Read market sessions, bars, snapshots, and reference metadata.
- Retrieve analyst estimates, earnings calendars, and corporate actions.
- Use rights-safe index rosters and constituents.

## Endpoints

<CardGroup cols={2}>
<Card title="GET /v1/market/bars" icon="square-terminal" href="/api-reference/market/get-v1-market-bars">Return historical OHLCV bars for a security with materialization and rights metadata</Card>
<Card title="GET /v1/market/calendar" icon="square-terminal" href="/api-reference/market/get-v1-market-calendar">Return market calendar sessions for supported exchanges with source coverage metadata</Card>
<Card title="GET /v1/market/corporate-actions" icon="square-terminal" href="/api-reference/market/get-v1-market-corporate-actions">Return rights-reviewed corporate actions used to normalize price history and event analysis</Card>
<Card title="GET /v1/market/earnings-calendar" icon="square-terminal" href="/api-reference/market/get-v1-market-earnings-calendar">Return upcoming and recent earnings events with consensus estimates, actuals, and surprise data</Card>
<Card title="GET /v1/market/estimates" icon="square-terminal" href="/api-reference/market/get-v1-market-estimates">Return analyst consensus ratings and price-target estimates for a ticker with provenance and freshness metadata</Card>
<Card title="GET /v1/market/indices" icon="square-terminal" href="/api-reference/market/get-v1-market-indices">Return the supported public index roster inventory and optionally include the broader rights-tracked index source catalog</Card>
<Card title="GET /v1/market/indices/constituents" icon="square-terminal" href="/api-reference/market/get-v1-market-indices-constituents">Return rights-safe index constituents for supported benchmark families with canonical entity mapping when available</Card>
<Card title="GET /v1/market/reference" icon="square-terminal" href="/api-reference/market/get-v1-market-reference">Return canonical market reference metadata for a security, including identifiers, exchange, and listing context</Card>
<Card title="GET /v1/market/search" icon="square-terminal" href="/api-reference/market/get-v1-market-search">Search for stock tickers by company name or keyword</Card>
<Card title="GET /v1/market/snapshots" icon="square-terminal" href="/api-reference/market/get-v1-market-snapshots">Return persisted latest-market snapshots for one or more securities with freshness, rights, and revision metadata</Card>
<Card title="GET /v1/market/universe" icon="square-terminal" href="/api-reference/market/get-v1-market-universe">Browse the full ticker universe with pagination, filtering by exchange, type, sector, industry, and search</Card>
</CardGroup>
