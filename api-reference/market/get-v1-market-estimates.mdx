# GET /v1/market/estimates

Return analyst consensus estimates and surprise context for a ticker from the Datastream market substrate.

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`
- `provenance`
- `freshness`
- `materialization`

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $OMNI_DATASTREAM_API_KEY" \
  -H "omni-version: 2026-03-19" \
  "https://api.secapi.ai/v1/market/estimates?ticker=AAPL&limit=3"
```
</RequestExample>

## Keep these fields

- estimate identity: `symbol`, `metric`, `fiscalPeriod`, `fiscalYear`
- decision fields: `value`, `actual`, `surprisePercent`, `analystCount`
- trust fields: `provenance`, `freshness`, `materialization`, `sourceRights`

## Failure posture

- treat non-2xx responses as contract-aware failures, not free-form errors
- preserve `requestId` and `traceparent` in logs and downstream reports
- if provenance or freshness metadata is present, return it unchanged so trust is not lost in the handoff
