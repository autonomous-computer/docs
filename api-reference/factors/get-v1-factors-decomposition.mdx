# GET /v1/factors/decomposition

Return factor-attribution decomposition for a security over a bounded lookback window with explained return, alpha, and methodology metadata

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $SECAPI_API_KEY" \
  -H "secapi-version: 2026-03-19" \
  "https://api.secapi.ai/v1/factors/decomposition?symbol=AAPL&window=3m&response_mode=compact&include=trust"
```
</RequestExample>
