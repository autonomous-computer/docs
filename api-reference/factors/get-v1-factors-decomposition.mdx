# GET /v1/factors/decomposition

Return factor-attribution decomposition for a security over a bounded lookback window with explained return, alpha, and methodology metadata

<Info>
Use this endpoint from server-side applications, SDKs, or agent workflows.
</Info>

## Response metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $SEC_API_KEY" \
  -H "secapi-version: 2026-03-19" \
  "https://api.secapi.ai/v1/factors/decomposition"
```
</RequestExample>
