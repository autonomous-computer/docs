# Facts API

<Info>
Read normalized SEC company facts by issuer, concept, unit, form, and period filters.
</Info>

## Use This API To

- Retrieve normalized XBRL facts for a company and concept.
- Build tag-level workflows when statements are too aggregated.
- Keep unit, form, and provenance details visible.
