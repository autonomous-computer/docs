---
title: "GET /v1/situations/issues"
description: "List immutable numbered weekly Special Situations Digest issues"
---

List the authenticated, paid archive of immutable weekly Special Situations Digest editions. Each result describes one numbered publication; retrieve the detail endpoint when an agent needs the frozen member snapshots and Markdown digest.

<Info>
Meter class: `situations_lookup`. This endpoint is authenticated and subject to the account's current API plan and limits.
</Info>

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $SECAPI_API_KEY" \
  -H "secapi-version: 2026-03-19" \
  "https://api.secapi.ai/v1/situations/issues?limit=12"
```
</RequestExample>

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `limit` | integer | Number of issues to return, from 1 through 100. |
