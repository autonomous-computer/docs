---
title: "GET /v1/situations/issues/{issue}"
description: "Retrieve one immutable weekly Special Situations Digest issue by number or slug"
---

Retrieve a frozen Special Situations Digest edition. The issue contains the publication metadata, Markdown digest, source IDs, and the exact SEC-derived situation snapshots included at publication time. Later lifecycle changes do not rewrite this record.

<Info>
Meter class: `situations_lookup`. This endpoint is authenticated and subject to the account's current API plan and limits.
</Info>

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $SECAPI_API_KEY" \
  -H "secapi-version: 2026-03-19" \
  "https://api.secapi.ai/v1/situations/issues/22"
```
</RequestExample>

The `{issue}` path segment accepts either a positive issue number or a slug such as `special-situations-digest-22-2026-07-05`.
