# Models API

<Info>
Analyze submitted model portfolios with factor attribution and hedge sections.

## Use This API To

- Analyze Model Builder or saved-model holdings without first persisting a model portfolio.
- Return factor exposure, attribution, hedge, and position-level views in one agent-friendly response.
- Preserve model identity and request metadata for downstream UI and agent workflows.
