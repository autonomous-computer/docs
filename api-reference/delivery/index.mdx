Inspect organization-scoped delivery records, export event history, or handle signed email unsubscriptions.

Start with [delivery events](/api-reference/delivery/get-v1-delivery-events) to inspect records for the current organization, then [export](/api-reference/delivery/get-v1-delivery-events-export) when you need a JSON or NDJSON copy. Configure outbound endpoints in the [Webhooks](/api-reference/webhooks) family and stream clients in [Streams](/api-reference/streams).

<Note>
The sidebar labels inside this family are the literal calls you make over HTTP. Descriptions stay in the body so the navigation stays sharp and scannable.
</Note>
