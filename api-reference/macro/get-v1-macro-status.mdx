# GET /v1/macro/status

Return macro source-health, country coverage, freshness, and source-rights posture from validation artifacts

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $SECAPI_API_KEY" \
  -H "secapi-version: 2026-03-19" \
  "https://api.secapi.ai/v1/macro/status"
```
</RequestExample>
