# POST /v1/stream_subscriptions/{stream_id}/disable

Disable a stream subscription without deleting its delivery records

<Info>
Audience: application and admin.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`

<RequestExample>
```bash
curl -X POST \
  -H "Authorization: Bearer $SECAPI_BEARER_TOKEN" \
  -H "secapi-version: 2026-03-19" \
  -H "content-type: application/json" \
  -d '{}' \
  "https://api.secapi.ai/v1/stream_subscriptions/strm_example_123/disable"
```
