Return recent Form N-PX proxy voting filings with cursor pagination and date filters

<Info>
Audience: application and coding agent.
</Info>

## Coverage and source scope

Use this route to locate filed Form N-PX proxy-voting disclosures for a fund or manager review. Results identify filings, not a complete current holdings list, an issuer-level vote tally, or an interpretation of voting intent. `date_from` and `date_to` bound filing dates; the filing's own reporting period may differ. A lower `submission_file_limit` can omit older SEC submission matches. Follow `nextCursor` with the same filters and retain the accession number for review. See [filing conventions](/api-conventions).

## Canonical metadata

- `requestId`
- `traceparent`
