# Forms API

<Info>
Retrieve specialized SEC form workflows such as Form 144 notices.
</Info>

## Use This API To

- List recent Form 144 notices of proposed insider sales.
- Filter notices with dates and pagination for monitoring workflows.
- Connect Form 144 activity to ownership or insider-trading analysis.
