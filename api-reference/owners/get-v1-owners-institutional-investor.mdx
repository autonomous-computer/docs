# GET /v1/owners/institutional/investor

Return an investor-centric institutional ownership graph for a manager CIK, including ranked holdings and quarter-over-quarter changes

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`
- `provenance`
- `freshness`
- `materialization`

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $SEC_API_KEY" \
  -H "secapi-version: 2026-03-19" \
  "https://api.secapi.ai/v1/owners/institutional/investor"
```
</RequestExample>

- preserve `requestId` and `traceparent` in logs and downstream reports
- if provenance or freshness metadata is present, return it unchanged so trust is not lost in the handoff
