# Ownership

Institutional and beneficial ownership surfaces for managers, holders, and ticker-centric analysis.

<Note>
The sidebar labels inside this family are the literal calls you make over HTTP. Descriptions stay in the body so the navigation stays sharp and scannable.
</Note>

## Operations in this family

<CardGroup cols={2}>
<Card title="GET /v1/owners/13d-13g" href="/api-reference/owners/get-v1-owners-13d-13g">List beneficial ownership reports across the SEC-native 13D and 13G filing families</Card>
<Card title="GET /v1/owners/13f" href="/api-reference/owners/get-v1-owners-13f">Return the latest normalized 13F ownership report for a manager CIK, with optional quarter-specific selection</Card>
<Card title="POST /v1/owners/13f/compare" href="/api-reference/owners/post-v1-owners-13f-compare">Compare the latest two 13F ownership reports for a manager CIK</Card>
<Card title="GET /v1/owners/13f/filings" href="/api-reference/owners/get-v1-owners-13f-filings">List recent 13F filings, including quarter-end report dates, for a manager CIK</Card>
<Card title="GET /v1/owners/institutional/extract" href="/api-reference/owners/get-v1-owners-institutional-extract">Return a quarter-specific historical institutional ownership extract for a manager CIK over the SEC-native 13F family</Card>
<Card title="GET /v1/owners/institutional/investor" href="/api-reference/owners/get-v1-owners-institutional-investor">Return an investor-centric institutional ownership graph for a manager CIK, including ranked holdings and quarter-over-quarter changes</Card>
<Card title="GET /v1/owners/institutional/ticker" href="/api-reference/owners/get-v1-owners-institutional-ticker">Return a ticker-centric institutional holder graph for the latest materialized 13F cohort, ranked by disclosed position value</Card>
</CardGroup>
