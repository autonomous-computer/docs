# Ownership API

<Info>
Read institutional and beneficial ownership APIs for managers, holders, and ticker-centric analysis.
</Info>

## Use This API To

- List beneficial ownership reports across 13D and 13G filings.
- Retrieve latest and historical 13F ownership reports.
- Compare holdings, inspect investors, and build ticker-centric ownership graphs.

## Endpoints

<CardGroup cols={2}>
<Card title="GET /v1/owners/13d-13g" icon="square-terminal" href="/api-reference/owners/get-v1-owners-13d-13g">List beneficial ownership reports across the SEC-native 13D and 13G filing families</Card>
<Card title="GET /v1/owners/13f" icon="square-terminal" href="/api-reference/owners/get-v1-owners-13f">Return the latest normalized 13F ownership report for a manager CIK, with optional quarter-specific selection</Card>
<Card title="POST /v1/owners/13f/compare" icon="square-terminal" href="/api-reference/owners/post-v1-owners-13f-compare">Compare the latest two 13F ownership reports for a manager CIK</Card>
<Card title="GET /v1/owners/13f/filings" icon="square-terminal" href="/api-reference/owners/get-v1-owners-13f-filings">List recent 13F filings, including quarter-end report dates, for a manager CIK</Card>
<Card title="GET /v1/owners/institutional/extract" icon="square-terminal" href="/api-reference/owners/get-v1-owners-institutional-extract">Return a quarter-specific historical institutional ownership extract for a manager CIK over the SEC-native 13F family</Card>
<Card title="GET /v1/owners/institutional/investor" icon="square-terminal" href="/api-reference/owners/get-v1-owners-institutional-investor">Return an investor-centric institutional ownership graph for a manager CIK, including ranked holdings and quarter-over-quarter changes</Card>
<Card title="GET /v1/owners/institutional/ticker" icon="square-terminal" href="/api-reference/owners/get-v1-owners-institutional-ticker">Return a ticker-centric institutional holder graph for the latest materialized 13F cohort, ranked by disclosed position value</Card>
</CardGroup>
