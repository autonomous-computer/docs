# Filings API

<Info>
Search filings, retrieve filing manifests, download source documents, export filings, and extract sections.
</Info>

## Use This API To

- Search filing manifests with form, issuer, date, and accession filters.
- Retrieve latest or accession-specific filings.
- Extract sections or export filings into application-ready formats.

## Before You Call

<Steps>
  <Step title="Resolve the issuer if the user supplied a ticker or company name">
    Resolve the issuer if the user supplied a ticker or company name.
  </Step>
  <Step title="Use cursor pagination for search and list workflows">
    Use cursor pagination for search and list workflows.
  </Step>
  <Step title="Keep accession numbers and source URLs with extracted content">
    Keep accession numbers and source URLs with extracted content.
  </Step>
</Steps>

## Endpoints

<CardGroup cols={2}>
<Card title="GET /v1/filings" icon="square-terminal" href="/api-reference/filings/get-v1-filings">Search filing manifests with historical filters, accession lookup semantics, sorting, and cursor pagination</Card>
<Card title="GET /v1/filings/{accession_number}" icon="square-terminal" href="/api-reference/filings/get-v1-filings-accession-number">Retrieve a filing manifest by accession number from the materialized filing corpus</Card>
<Card title="GET /v1/filings/{accession_number}/download" icon="square-terminal" href="/api-reference/filings/get-v1-filings-accession-number-download">Redirect to the SEC EDGAR source document for a filing by accession number</Card>
<Card title="GET /v1/filings/{accession_number}/export" icon="square-terminal" href="/api-reference/filings/get-v1-filings-accession-number-export">Export a filing as JSON, Markdown, CSV, XLSX, DOCX, or PDF</Card>
<Card title="GET /v1/filings/{accession_number}/sections/{section_key}" icon="square-terminal" href="/api-reference/filings/get-v1-filings-accession-number-sections-section-key">Extract a filing item or section from a specific accession-number filing</Card>
<Card title="GET /v1/filings/latest" icon="square-terminal" href="/api-reference/filings/get-v1-filings-latest">Retrieve the latest filing for an entity and form</Card>
<Card title="GET /v1/filings/latest/render" icon="square-terminal" href="/api-reference/filings/get-v1-filings-latest-render">Render the latest filing into Markdown-like text</Card>
<Card title="GET /v1/filings/latest/risk-categories" icon="square-terminal" href="/api-reference/filings/get-v1-filings-latest-risk-categories">Return deterministic Item 1A risk-category coverage for the latest covered filing of an issuer</Card>
<Card title="GET /v1/filings/latest/sections/{section_key}" icon="square-terminal" href="/api-reference/filings/get-v1-filings-latest-sections-section-key">Extract a section from the latest filing for an entity and form</Card>
<Card title="GET /v1/filings/pension-benefit-schedule" icon="square-terminal" href="/api-reference/filings/get-v1-filings-pension-benefit-schedule">Return structured expected pension and retiree benefit payments for a target year when the filing discloses the schedule in rendered tables</Card>
