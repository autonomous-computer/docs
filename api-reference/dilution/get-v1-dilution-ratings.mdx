## Derived rating history

Ratings are derived analytical outputs, not SEC disclosures or forecasts. This list reads stored rating history and can be incomplete where history has not been materialized. Preserve `numericScore`, `overallRisk`, `factors`, `verification.confidence`, freshness, and provenance; use [GET /v1/dilution/score](/api-reference/dilution/get-v1-dilution-score) for the current calculation. Read the [Dilution Score guide](/dilution-score) before interpreting a returned `overallRisk`.
