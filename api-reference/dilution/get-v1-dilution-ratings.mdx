# GET /v1/dilution/ratings

Composite Dilution Score with sub-factors (offering ability, historical, cash need, warrant exercise risk). Ordered by `computed_at desc`. Verification block on every row.

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`
- `verification`
- `provenance`

## Query parameters

| Param | Type | Description |
|---|---|---|
| `ticker` | string | Issuer ticker. |
| `overall_risk` | string | Filter (`low`, `medium`, `high`). |
| `cursor` | int | From prior page's `nextCursor`. |
| `limit` | int (1–50, default 10) | Page size. |
| `view` | enum | Response shape. |

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $OMNI_DATASTREAM_API_KEY" \
  -H "omni-version: 2026-03-19" \
  "https://api.secapi.ai/v1/dilution/ratings?overall_risk=high"
```
</RequestExample>
