## Evidence and limits

Statuses summarize disclosed notice and remediation context in covered records; they are not an exchange-authoritative, real-time listing feed. Preserve the source date and filing reference, and confirm material current status with the issuer or exchange. See [freshness and trust](/freshness-and-trust).
