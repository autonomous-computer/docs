# Dilution API

<Info>
Track offering pressure, warrants, convertibles, lockups, float changes, and issuer dilution risk.
</Info>

## Use This API To

- Screen issuers by dilution events, ratings, or score.
- Inspect warrants, convertibles, ROFR terms, lockups, and cash runway signals.
- Track share-float history, reverse splits, and Nasdaq compliance events.

## Before You Call

<Steps>
  <Step title="Start with a ticker or issuer identifier">
    Start with a ticker or issuer identifier.
  </Step>
  <Step title="Use view=agent for compact list responses when supported">
