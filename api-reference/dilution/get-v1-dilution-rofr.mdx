# GET /v1/dilution/rofr

List right-of-first-refusal and tail-financing clauses parsed from underwriter agreements. Verification block on every row.

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`
- `verification`
- `provenance`

## Query parameters

| Param | Type | Description |
|---|---|---|
| `ticker` | string | Issuer ticker. |
| `cik` | string | SEC CIK. |
| `accession_number` | string | Match a specific filing. |
| `filed_at_from` / `filed_at_to` | ISO date | Bounds on `filed_at`. |
| `cursor` | int | From prior page's `nextCursor`. |
| `limit` | int (1–50, default 10) | Page size. |
| `view` | enum | Response shape. |

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $OMNI_DATASTREAM_API_KEY" \
  -H "omni-version: 2026-03-19" \
  "https://api.secapi.ai/v1/dilution/rofr?ticker=ACME"
```
</RequestExample>

## Give this prompt to your agent

<Prompt>
Use Omni Datastream `GET /v1/dilution/rofr` to surface right-of-first-refusal and tail-financing exposure. Preserve `requestId`, `traceparent`, and the `verification` block. Flag any clause where `rightOfFirstRefusalPresent` or `tailFinancingPaymentsPresent` is true and the duration extends past the next anticipated capital raise.
</Prompt>

## Failure posture

- treat non-2xx responses as contract-aware failures, not free-form errors
- preserve `requestId` and `traceparent`
