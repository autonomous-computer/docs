# GET /v1/dilution/corporate-actions

List dilution-relevant corporate actions: ticker changes, exchange moves, de-SPAC closings, splits. Ordered by `effective_date desc`. Verification block on every row.

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`
- `verification`
- `provenance`

## Query parameters

| Param | Type | Description |
|---|---|---|
| `ticker` | string | Issuer ticker. |
| `action_type` | string | Filter (`ticker_change`, `exchange_move`, `despac`, `split`, etc.). |
| `effective_date_from` / `effective_date_to` | ISO date | Bounds on `effective_date`. |
| `cursor` | int | From prior page's `nextCursor`. |
| `limit` | int (1–50, default 10) | Page size. |
| `view` | enum | Response shape. |

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $OMNI_DATASTREAM_API_KEY" \
  -H "omni-version: 2026-03-19" \
  "https://api.secapi.ai/v1/dilution/corporate-actions?ticker=ACME"
```
</RequestExample>
