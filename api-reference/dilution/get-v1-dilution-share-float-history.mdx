# GET /v1/dilution/share-float-history

Historical shares outstanding and public float series materialized from filings. Ordered by `as_of_date desc`. Verification block on every row.

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`
- `verification`
- `provenance`

## Query parameters

| Param | Type | Description |
|---|---|---|
| `ticker` | string | Issuer ticker. |
| `as_of_date_from` / `as_of_date_to` | ISO date | Bounds on `as_of_date`. |
| `cursor` | int | From prior page's `nextCursor`. |
| `limit` | int (1–50, default 10) | Page size. |
| `view` | enum | Response shape. |

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $OMNI_DATASTREAM_API_KEY" \
  -H "omni-version: 2026-03-19" \
  "https://api.secapi.ai/v1/dilution/share-float-history?ticker=ACME&limit=4"
```
</RequestExample>

## Give this prompt to your agent

<Prompt>
Use Omni Datastream `GET /v1/dilution/share-float-history` to chart shares-outstanding and public-float deltas over time. Preserve `requestId`, `traceparent`, and `verification`. Compute period-over-period dilution as `(sharesOutstanding[t] - sharesOutstanding[t-1]) / sharesOutstanding[t-1]`.
</Prompt>

## Failure posture

- treat non-2xx responses as contract-aware failures, not free-form errors
- preserve `requestId` and `traceparent`
