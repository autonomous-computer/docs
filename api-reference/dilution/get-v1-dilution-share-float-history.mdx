<Info>
Audience: investors and developers reconciling disclosed share counts and public-float observations.
</Info>

- `data`

## Evidence and limits

Rows preserve reported shares outstanding and, where available, disclosed public float. Public float is a value-based disclosure and is not interchangeable with a share count; missing rows are not zeros. Use observation date, source form, accession, and verification when building a history. See [freshness and trust](/freshness-and-trust).

## Failure posture

- keep as-of date, units, source form, accession, and verification
- never label a shares-outstanding value as public float
- treat gaps as unavailable coverage rather than zero float or zero shares
