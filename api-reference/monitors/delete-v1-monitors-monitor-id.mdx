# DELETE /v1/monitors/{monitor_id}

Deactivate a monitor (idempotent; sets is_active=false)

<Info>
Use this endpoint from server-side applications, SDKs, or agent workflows.
</Info>

## Response metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
curl -X DELETE \
  -H "x-api-key: $SEC_API_KEY" \
  -H "secapi-version: 2026-03-19" \
  -H "content-type: application/json" \
  -d '{}' \
  "https://api.secapi.ai/v1/monitors/{monitor_id}"
```
</RequestExample>
