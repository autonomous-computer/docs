# GET /v1/statements/share-float

Return a share-float wrapper backed by SEC company facts, including disclosed public float when available and shares-outstanding fallback semantics otherwise

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`
- `provenance`
- `freshness`
- `materialization`
- `capability`
- `sourceMode`

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $SECAPI_API_KEY" \
  -H "secapi-version: 2026-03-19" \
  "https://api.secapi.ai/v1/statements/share-float?symbol=MSFT"
```
</RequestExample>

## Query parameters

- `ticker`, `symbol`, or `cik` identifies the issuer. `symbol` is accepted as an alias for `ticker`.

## Failure posture

- treat non-2xx responses as contract-aware failures, not free-form errors
- preserve `requestId` and `traceparent` in logs and downstream reports
- if provenance or freshness metadata is present, return it unchanged so trust is not lost in the handoff
