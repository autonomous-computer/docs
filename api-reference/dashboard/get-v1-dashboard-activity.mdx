# GET /v1/dashboard/activity

Return dashboard activity totals and recent request rows for the current organization

<Info>
Audience: application and coding agent.
</Info>

## Canonical metadata

- `requestId`
- `traceparent`

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "Cookie: omni_datastream_session=$SECAPI_WORKOS_SESSION_COOKIE" \
  -H "secapi-version: 2026-03-19" \
  "https://api.secapi.ai/v1/dashboard/activity"
```
</RequestExample>

<Info>
Dashboard endpoints require the `omni_datastream_session` WorkOS browser-session cookie. API keys are rejected for dashboard control-plane access.
</Info>
