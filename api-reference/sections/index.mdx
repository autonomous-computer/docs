# Sections API

<Info>
Search and extract filing sections, items, and snippets.
</Info>

## Use This API To

- Search filing sections with filters and cursor pagination.
- Retrieve section-level snippets for diligence, monitoring, or agent workflows.
- Connect section hits back to source filings and accession numbers.
