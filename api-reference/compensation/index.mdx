# Compensation API

<Info>
Retrieve and compare executive compensation disclosures from proxy filings.
</Info>

## Use This API To

- Read normalized executive compensation from DEF 14A disclosures.
- Compare the latest two compensation disclosures for an issuer.
- Connect pay data to governance, board, and ownership workflows.
