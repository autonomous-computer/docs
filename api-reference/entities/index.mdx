# Entities API

<Info>
Resolve and search canonical public-market entities across issuers, managers, insiders, and funds.
</Info>

## Use This API To

- Resolve tickers, symbols, CIKs, CUSIPs, ISINs, FIGI-family identifiers, and names.
- Search canonical SEC entities before calling filing or ownership endpoints.
- Use confidence and match-basis metadata for disambiguation.
