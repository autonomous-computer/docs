# Companies

Reference pages for canonical company fundamentals endpoints.

<Note>
The `/v1/companies/*` family is the canonical public surface for company financial data. It unifies SEC XBRL statements with MASSIVE-backed financials and provides lineage metadata for source attribution.
</Note>

## Operations in this family

<CardGroup cols={2}>
<Card title="GET /v1/companies/financials" href="/api-reference/companies/get-v1-companies-financials">Return unified financial statements (income, balance sheet, cash flow) for a company</Card>
<Card title="GET /v1/companies/ratios" href="/api-reference/companies/get-v1-companies-ratios">Return computed financial ratios (profit margin, ROA, ROE, D/E, current ratio) for a company</Card>
<Card title="GET /v1/companies/income-statements" href="/api-reference/companies/get-v1-companies-income-statements">Return income statements backed by SEC XBRL filings</Card>
<Card title="GET /v1/companies/balance-sheets" href="/api-reference/companies/get-v1-companies-balance-sheets">Return balance sheets backed by SEC XBRL filings</Card>
<Card title="GET /v1/companies/cash-flow-statements" href="/api-reference/companies/get-v1-companies-cash-flow-statements">Return cash flow statements backed by SEC XBRL filings</Card>
</CardGroup>
