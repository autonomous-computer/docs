# GET /v1/companies/ratios

Return computed financial ratios for a company including profit margin, return on assets, return on equity, debt-to-equity, and current ratio

<Info>
Audience: application and coding agent.
</Info>

## Query parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `ticker` | string | Stock ticker symbol (required) |
| `period` | string | `annual` or `quarterly` (default: `annual`) |
| `limit` | integer | Number of periods to return (default: 4, max: 50) |

## Canonical metadata

- `requestId`
- `traceparent`
- `lineage` — source attribution (`massive_financials`)

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $OMNI_DATASTREAM_API_KEY" \
  -H "omni-version: 2026-03-19" \
  "https://api.secapi.ai/v1/companies/ratios?ticker=AAPL&period=annual&limit=4"
```
</RequestExample>
