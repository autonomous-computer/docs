# GET /v1/companies/income-statements

Return income statements for a company backed by SEC XBRL filings

<Info>
Audience: application and coding agent.
</Info>

## Query parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `ticker` | string | Stock ticker symbol (e.g., AAPL) |
| `cik` | string | SEC CIK number |
| `period` | string | `annual` or `quarterly` (default: `annual`) |
| `limit` | integer | Number of periods to return (default: 4, max: 50) |
| `view` | string | `compact` for reduced payload |

## Canonical metadata

- `requestId`
- `traceparent`
- `lineage` — source attribution (`sec_xbrl`)

## Example request

<RequestExample>
```bash
curl -X GET \
  -H "x-api-key: $OMNI_DATASTREAM_API_KEY" \
  -H "omni-version: 2026-03-19" \
  "https://api.secapi.ai/v1/companies/income-statements?ticker=AAPL&period=annual&limit=4"
```
</RequestExample>
