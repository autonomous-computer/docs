# Model Portfolios API

<Info>
Inspect model portfolio factor views and position-level exposure context.
</Info>

## Use This API To

- Retrieve aggregate model-portfolio factor views.
- Inspect per-position exposures for drill-down workflows.
- Connect model portfolios to factor or strategy analysis.
- Use `/v1/models/factor-analysis` when you want ad hoc Model Builder holdings analyzed without first persisting a model portfolio.
