# Python SDK

## What it is good for

- notebooks
- research pipelines
- ETL and enrichment services
- backend jobs that want compact REST parity without hand-rolling requests

## First useful calls

```python
from omni_datastream_py import OmniDatastreamClient

client = OmniDatastreamClient(
    api_key="ods_test_...",
    base_url="https://api.secapi.ai",
)

issuer = client.resolve_entity(ticker="AAPL")
filing = client.latest_filing(ticker="AAPL", form="10-K")
statements = client.all_statements(ticker="AAPL", period="annual", limit=1)
calendar = client.market_calendar(market="XNYS", duration=5)
volatility = client.volatility_signal(ticker="AAPL")
```
