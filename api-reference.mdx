# API Reference

Generated from the canonical Omni Datastream wire contract.

## Conventions

- Base URL: `https://api.secapi.ai`
- Version header: `omni-version: 2026-03-13`
- Authentication: `x-api-key` for REST requests
- Response envelope: every public response carries `requestId`; data-heavy responses also carry provenance and freshness metadata where applicable
- Agent handoff: every endpoint block includes a copyable prompt you can pass to a coding agent
