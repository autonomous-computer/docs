# Pay-As-You-Go Cost Calculator

SEC API offers pay-as-you-go pricing with no monthly minimum. Prices below use
the `standard_reads` meter at `$0.02` per call from the `GET /v1/billing/rates`
catalog verified on 2026-04-27.

## Pricing Differences vs sec-api.io

- **PAYG:** sec-api.io's public self-serve pricing is fixed monthly subscription-based.
- **Meter transparency:** SEC API exposes per-meter prices through `GET /v1/billing/rates`.
- **Per-call billing:** SEC API exposes per-meter prices through `GET /v1/billing/rates`.
- **Budget controls:** SEC API exposes quote and budget endpoints for spend controls.
- **Intelligence bundles:** SEC API packages common agent briefings into single calls.
