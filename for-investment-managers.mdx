## Key endpoints for investment workflows

| Endpoint | Method | Use case |
|----------|--------|----------|
| `/v1/entities/resolve` | `GET` | Resolve ticker, CIK, or name before research reads |
| `/v1/factors/exposures` | `GET` | Per-stock factor exposure loadings |
| `/v1/factors/returns` | `GET` | Historical factor return series with z-scores |
| `/v1/factors/decomposition` | `GET` | Factor decomposition for a single ticker |
| `/v1/intelligence/security` | `GET` | Comprehensive single-security analysis bundle |
| `/v1/intelligence/portfolio` | `POST` | Portfolio-level macro and factor analysis |
| `/v1/owners/13f` | `GET` | Latest 13F holdings report for a manager filer |
| `/v1/owners/13f/compare` | `POST` | Quarter-over-quarter 13F position changes |
| `/v1/owners/institutional/ticker` | `GET` | Issuer-level institutional holder lookup |
| `/v1/insiders` | `GET` | Insider transactions by company or executive |

## Example: track manager-level ownership changes

13F routes are keyed by the manager filer's CIK, not the issuer ticker. For Berkshire Hathaway:

```bash
curl -X POST \
  -H "content-type: application/json" \
  -H "x-api-key: $SECAPI_API_KEY" \
  -d '{"cik":"0001067983","limit":20}' \
  "https://api.secapi.ai/v1/owners/13f/compare"
```

If you need issuer-level holders, use the institutional-holder route instead:

```bash
curl -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/owners/institutional/ticker?ticker=AAPL&limit=25"
```

Then cross-check with recent insider activity for the issuer:

```bash
curl -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/insiders?ticker=AAPL&forms=4&limit=10"
```

## Example: factor and portfolio context

Pull current factor exposures for a ticker in your portfolio:

```bash
curl -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/factors/exposures?symbols=NVDA&response_mode=compact"
```

Package a portfolio-level intelligence read for an internal research note:

```bash
curl -X POST \
  -H "content-type: application/json" \
  -H "x-api-key: $SECAPI_API_KEY" \
  -d '{"holdings":[{"symbol":"AAPL","weight":0.25},{"symbol":"MSFT","weight":0.25},{"symbol":"NVDA","weight":0.25},{"symbol":"GOOGL","weight":0.25}]}' \
  "https://api.secapi.ai/v1/intelligence/portfolio"
```
