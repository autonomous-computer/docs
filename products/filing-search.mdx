---
title: "SEC filing search"
description: "Find SEC filing manifests by issuer, form, date, fiscal period, accession number, or indexed text before retrieval and extraction."
---

Use filing search to identify the exact SEC filing your application will retrieve, extract, or cite. Save its accession number as the stable handoff between steps.

## What you can do

- Find filing manifests by ticker, `symbol`, CIK, form, date range, fiscal period, calendar filing year, or issuer fiscal year.
- Look up an accession number, search indexed filing-manifest text with `q`, and choose ascending or descending filing-date order.
- Fetch a selected filing, or extract a named section from that accession.
- Retrieve the latest filing or latest named section when historical accession selection is not required.

## Endpoints

| Method | Path | What it returns |
| --- | --- | --- |
| `GET` | [`/v1/filings`](/api-reference/filings) | A paginated filing-manifest list. |
| `GET` | [`/v1/filings/{accession_number}`](/api-reference/filings) | One filing manifest. |
| `GET` | [`/v1/filings/{accession_number}/sections/{section_key}`](/api-reference/filings) | A named section from that filing. |
| `GET` | [`/v1/filings/latest`](/api-reference/filings/get-v1-filings-latest) | The latest filing matching an issuer and optional form. |
| `GET` | [`/v1/filings/latest/sections/{section_key}`](/api-reference/filings/get-v1-filings-latest-sections-section-key) | A named section from the latest filing matching an issuer and form. |

## Example

```bash
curl -sS \
  -H "x-api-key: $SECAPI_API_KEY" \
  "https://api.secapi.ai/v1/filings?ticker=AAPL&form=10-K&sort=filing_date_desc&limit=5"
```

The list response includes the filing identity needed for later calls, including `accessionNumber`, form, filing date, and SEC filing URL. Keep `requestId` and any provenance, freshness, or materialization fields returned with the record.

## Boundaries to keep in view

- `ticker`, `symbol`, and CIK must not conflict. Resolve ambiguous input before filing-specific use.
- `limit` is 1 through 50 and `cursor` is a zero-based offset. Do not reuse a cursor after changing filters.
- Filing availability follows SEC source material. A missing result is a retrieval outcome, not proof that an issuer made no filing.
- Use the source filing URL and accession number for a conclusion that needs source review; a manifest search result is not the filing text.
